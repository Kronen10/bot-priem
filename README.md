# bot-priem
Данный телеграмм-бот разрабатывался специально для работы приемной комиссии ЧИ БГУ. Предназначен для консультации абитуриентов по самым популярным вопросам.

## Установка и запуск
Для работы бота нужно установить библиотеку telebot c помощью команды "pip install telebot".
И запустить с помощью команды python bot.py
